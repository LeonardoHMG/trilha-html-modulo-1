História dos Computadores

(Voltar)

A história dos computadores remonta ao século XIX, com a criação das primeiras máquinas que tentavam automatizar cálculos.

Charles Babbage é considerado o "pai do computador" por ter projetado a Máquina Analítica, que possuía conceitos essenciais dos computadores modernos, como a unidade de controle e a memória. No entanto, foi apenas no século XX que os computadores se tornaram viáveis para uso geral.

O desenvolvimento de computadores eletrônicos começou na década de 1930 e 1940, com a invenção do ENIAC durante a Segunda Guerra Mundial, um dos primeiros computadores eletrônicos de grande escala.

Ao longo das décadas seguintes, os computadores passaram por várias gerações de evolução, com avanços em transistores, circuitos integrados e, posteriormente, microprocessadores, que permitiram a miniaturização e a popularização da tecnologia.

A partir dos anos 1970 e 1980, o uso de computadores se expandiu para empresas e, mais tarde, para os consumidores individuais, com a popularização de computadores pessoais, como o Apple II e o IBM PC.

Surgimento da Internet

(Voltar)

A Internet teve seu surgimento na década de 1960, como uma rede de comunicação entre computadores em universidades e laboratórios de pesquisa. O projeto inicial, chamado ARPANET, foi financiado pela DARPA e visava criar uma rede descentralizada para compartilhar informações e pesquisa científica.

Nos anos seguintes, a ARPANET evoluiu e, com o desenvolvimento de protocolos de comunicação, como o TCP/IP, ela se expandiu e se transformou na rede global que conhecemos hoje como a Internet.

O grande avanço que possibilitou a popularização da Internet foi a criação da World Wide Web (WWW) por Tim Berners-Lee, em 1991. A partir daí, os navegadores da web se tornaram ferramentas acessíveis para milhões de pessoas em todo o mundo, e a Internet passou a ser usada não só para a comunicação e pesquisa científica, mas também para fins comerciais, sociais e de entretenimento.

O que são Clients?

(Voltar)

Em termos de redes e computação, "client" (ou "cliente") refere-se a qualquer dispositivo ou programa que acessa os serviços de um servidor em uma rede. O cliente solicita informações ou serviços do servidor e aguarda uma resposta. Por exemplo, em uma rede de computadores, o navegador de internet de um usuário é um cliente que acessa a web (servidor), solicitando páginas e dados.

Em sistemas distribuídos, como a web, os clientes podem ser computadores pessoais, dispositivos móveis, ou aplicativos específicos que utilizam a rede para se conectar a servidores e utilizar os recursos compartilhados.

O que são Servers?

(Voltar)

Um "server" (ou "servidor") é um computador ou software que fornece serviços ou recursos a outros computadores (clientes) em uma rede. O servidor espera por solicitações de clientes e responde a elas, seja fornecendo dados, processando informações ou gerenciando tarefas. Por exemplo, em uma aplicação web, o servidor armazena páginas HTML, imagens e outros recursos, e envia esses conteúdos para os clientes (usuários que acessam o site).

Existem diferentes tipos de servidores, incluindo servidores de arquivos, servidores de banco de dados, servidores web e servidores de e-mail, cada um com uma função específica na rede.

Estrutura básica HTML e Tags

(Voltar)

HTML é a linguagem padrão para criar páginas web. A estrutura básica de um documento HTML consiste em elementos que indicam como o conteúdo deve ser exibido em um navegador. A estrutura mínima de um documento HTML inclui:

  • <!DOCTYPE html>: Declaração do tipo de documento, que informa ao navegador que o arquivo é um documento HTML.
  • <html>: Elemento raiz do documento.
  • <head>: Contém metadados sobre a página, como título, links para folhas de estilo, scripts, etc.
  • <title>: Define o título da página, que aparece na aba do navegador.
  • <body>: Contém o conteúdo visível da página, como texto, imagens, links, etc.
  • Dentro do <body>, é possível usar tags para estruturar o conteúdo, como parágrafos (<p>), títulos (<h1> até <h6>), listas (<ol>, <ul> e <li>), links (<a>), imagens (<img>), entre outros.

O que são Tags:

No contexto do HTML, tags são elementos usados para marcar o conteúdo dentro de um documento web. Cada tag é envolvida por colchetes angulares < e >. As tags indicam ao navegador como o conteúdo deve ser interpretado e exibido. Existem tags de abertura e fechamento, como <p> para um parágrafo e </p> para fechar o parágrafo.

Existem também tags auto-fechantes, como <img />, que não necessitam de uma tag de fechamento, porque o elemento é completo na própria tag de abertura.

Atributos das Tags:

Atributos em tags HTML são usados para fornecer informações adicionais ou configurar aspectos específicos de um elemento. Eles geralmente aparecem dentro da tag de abertura e possuem um nome e um valor. Por exemplo, o atributo href em uma tag <a> define o destino de um link, como no caso de <a href="https://www.exemplo.com">Clique aqui</a>. Outros exemplos de atributos incluem src (para especificar a origem de uma imagem), alt (para fornecer uma descrição alternativa para uma imagem) e class (para associar a um estilo CSS)

Textos, Listas Ordenadas, Não Ordenadas e Links

(Voltar)

Textos:

No HTML, o texto pode ser formatado de várias maneiras usando tags específicas. Por exemplo:

  • <p>: Define um parágrafo.
  • <h1> a <h6>: São tags de títulos, sendo <h1> o título mais importante e <h6> o menos importante.
  • <strong>: Marca um texto como importante, geralmente exibido em negrito.
  • <em>: Marca um texto como enfatizado, geralmente exibido em itálico.
  • <u>: Marca um texto com sublinhado.

Listas Ordenadas e Não Ordenadas:

As listas no HTML podem ser ordenadas ou não ordenadas:

  • Listas ordenadas (<ol>): São listas numeradas ou com uma ordem específica. Cada item da lista é definido com a tag <li> (list item).
  • Listas não ordenadas (<ul>): São listas sem uma ordem específica, geralmente apresentadas com marcadores (bolinhas). Também usam a tag <li> para definir cada item.

Links:

Em HTML, os links são criados usando a tag <a> (ancora). Esta tag permite que os usuários naveguem entre diferentes páginas ou recursos ao clicar em um texto ou imagem. O atributo href é usado para especificar o endereço (URL) de destino do link.
